# Contact-List
A simple Contact List Web App made using Express JS

# How to use
* Get NodeJs on your System.
* Clone this repository .
* Open in terminal/cmd
* Do npm install
* Go to Web Browser and hit https://localhost:8000

# Made using
* Node Js
* Express
* MongoDB
